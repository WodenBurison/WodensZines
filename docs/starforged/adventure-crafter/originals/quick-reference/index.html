Quick Reference

Condensed reference for running The Adventure Crafter, as-you-go / as-needed style. Full rules and examples are in the PDF if you need more than this.

1. Set your Themes (once per Adventure)

Every Adventure runs on 5 Themes — Action, Tension, Mystery, Social, Personal — ranked 1st (most likely to come up) through 5th (least likely). Edit the Theme names directly into Theme Priority Roll in that order (swap them around per Adventure as needed) — either choose the order yourself, or roll Random Themes Table once per priority slot (re-roll repeats) to fill it in randomly.

2. Start a Turning Point

Roll 1d100 on Plotlines List to see which Plotline this Turning Point is about.

  • Choose Most Logical Plotline → an existing Plotline is Invoked (this becomes a Development, or a Conclusion if the first Plot Point rolled says so).
  • New Plotline → you're starting a fresh Plotline; name it once the Turning Point's Plot Points make its shape clear.

3. Roll 2–5 Plot Points

For each Plot Point:

  1. Roll 1d20 on Theme Priority Roll — it tells you the Theme directly and links straight to that Theme's Plot Points table (living in Subtables).
  2. Roll 1d100 on the linked table to get your Plot Point.
  3. A result of Meta sends you to Meta Plot Points instead (changes the Characters List or combines Plotlines rather than describing an event).
  4. A result of None means no Plot Point this slot — leave it blank.
  5. A result of Conclusion ends the Plotline for this Turning Point.

Every Turning Point needs 2–5 Plot Points, and no more than three "None" results (re-roll a 4th).

4. Invoke Characters

Whenever a Plot Point calls for a Character, roll 1d100 on Characters List.

  • Choose Most Logical Character → use an existing Character.
  • New Character → craft one (see below), then write them into Characters List and next to the Plot Point on your Adventure Sheet.

5. Craft a new Character (only when "New Character" comes up)

  1. Roll 1d100 on Character Special Trait — individual/organization/object, connected/not to this Plotline, helps/hinders, or connected to an existing Character.
  2. Roll 1d100 on Character Identity (roll twice and combine on a 1–33).
  3. Roll 1d100 on Character Descriptors (roll twice and combine on a 1–21).
  4. Combine with the Plot Point and Turning Point context, name them, and write them down.

6. Interpret and move on

Take the Plotline status, Plot Points, and any Invoked Characters, and turn them into what actually happens. Treat results you can't make sense of as "I Dunno" — reroll or ignore rather than forcing it.
